# 🍪 LinkedIn Cookie Extractor - Installation & Usage

## 📦 What's Included
- `linkedin-cookie-extractor.zip` - The Chrome extension
- Complete source code for transparency

## 🚀 Installation Steps

### 1. Download & Extract
- Download `linkedin-cookie-extractor.zip` 
- Extract it to a folder (like `linkedin-cookie-extractor/`)

### 2. Install in Chrome
1. Open Chrome and go to: `chrome://extensions/`
2. Turn ON "Developer mode" (top right toggle)
3. Click "Load unpacked" 
4. Select the `linkedin-cookie-extractor` folder
5. The extension should appear with a LinkedIn/cookie icon

## 🎯 How to Use

### Step 1: Login to LinkedIn
1. Go to `linkedin.com` in Chrome
2. Login to your account normally
3. Make sure you're fully authenticated

### Step 2: Extract Cookies
1. Click the extension icon in Chrome toolbar (🍪)
2. Click "Extract LinkedIn Cookies" 
3. The extension will scan and extract all LinkedIn session cookies
4. Copy the JSON output

### Step 3: Share with kkbot
1. Copy the cookie JSON data
2. Send it to kkbot in WhatsApp
3. kkbot can then use these cookies for LinkedIn automation

## 🔒 Security Notes
- **Cookies are sensitive** - only share with trusted automation
- **Session expires** - may need to re-extract cookies periodically  
- **Private browsing** - won't work in incognito mode
- **Only LinkedIn** - extension only accesses linkedin.com cookies

## 🛠️ What the Extension Does
- Reads cookies from `*.linkedin.com` domains
- Formats them in JSON for easy automation use
- Includes session data, CSRF tokens, authentication cookies
- Provides copy-to-clipboard functionality

## ❌ Troubleshooting
- **No cookies found**: Make sure you're logged into LinkedIn
- **Extension not working**: Check Developer mode is ON
- **Can't install**: Ensure you extracted the ZIP file first
- **Outdated cookies**: Re-extract if sessions expire
